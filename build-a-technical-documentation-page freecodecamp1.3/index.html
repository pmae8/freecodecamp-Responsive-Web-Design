Introduction

Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically-typed and garbage-collected.

Python supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.

Variables and Data Types

Variables are used to store data values. Python has various data types including:

  • Numeric Types: int, float, complex
  • Text Type: str
  • Boolean Type: bool
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
# Example name = "John Doe" age = 30 is_student = True

Operators

Operators are used to perform operations on variables and values.

Arithmetic operators: +, -, *, /, //, %, **

Comparison operators: ==, !=, >, <, >=, <=

Logical operators: and, or, not

# Example sum = 10 + 5 is_equal = 10 == 10

Control Flow

Control flow statements are used to control the execution of code.

Conditional Statements: if, elif, else

Loops: for, while

# Example if age >= 18: print("You are an adult") else: print("You are a minor")

Functions

A function is a block of code which only runs when it is called.

# Example def greet(name): print("Hello, " + name + "!") greet("John")

Data Structures

Data structures are used to store collections of data.

Lists: ordered, changeable, allow duplicate values.

Tuples: ordered, unchangeable, allow duplicate values.

Dictionaries: unordered, changeable, indexed, no duplicate members.

# Example my_list = ["apple", "banana", "cherry"] my_tuple = ("apple", "banana", "cherry") my_dict = {"name": "John", "age": 30}
